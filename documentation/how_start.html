How to start? - LMS

  • Created : 27 September 16

In order for you to start using LMS make sure you have created the following options:


1. Create Classes.

2. Create Sections.

3. Create Subjects.

4. Create Transport.

5. Create Classrooms.

6. Setup your Timezone.

7. Setup your Running Year.

8. Upload your Logotype, Favicon & Background.


Now you can create your users:


1. Create Admins.

2. Create Teachers.

3. Create Accountants.

4. Create Librarians.

5. Create Parents.

6. Create Students.
