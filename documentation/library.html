Library - LMS

  • Created : 27 September 16

To manage your library you must perform the following steps:


* Go to Library in the navigation menu.


* In the library you can upload books in any format, both downloadable books and physical books. Students will be able to view the books in their account.

Both the administrator and the librarian will be able to view book requests and will be able to approve and reject them.

To add a new book click on the green button and fill in the necessary fields.
