Admin permissions - LMS

  • Created : 27 September 16

To update admin permission you must perform the following steps:


* Go to Usuers in the navigation menu (Available only for Super Admin).


Click on the 3 points about Admins and then Permissions.

Mark the permissions you want the admin account to have and click on update.
